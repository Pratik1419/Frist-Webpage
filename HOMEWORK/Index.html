HTML & CSS Basics Summary

HTML Summary

HTML stands for (Hyper Text Markup Language). It is the standard markup language for documents designed to be displayed in a web browser. you don't use it for styling purpose.

  • HTML documents are made of HTML elements.
  • HTML elements are the building blocks of HTML documents.
  • HTML elements typically have an opening tag,content and then a closing tag
  • You can also void (empty) elements likes image
  • There's a long list of available elements but you'll gain experience over time. No worried

Learn more about all available HTML elements onMDN HTML elements reference

CSS Summary

CSS stands for (Cascading Style Sheets). It is the stylesheet language for documents written in HTML.

  • Style are assigned via property-value pairs
  • CSS styles are applied to HTML elements
  • CSS styles are usually assigned to HTML elements
  • You can assign styles via the "style" attribute
  • Alternatively, you can work with external stylesheet files which you "link" to

learn more about all available CSS elements onMDN CSS property reference
